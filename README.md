# ​🚀​ EstacIA

O EstacIA é uma aplicação web voltada para o gerenciamento inteligente de estacionamentos em shoppings, utilizando tecnologia para otimizar o fluxo de veículos de ponta a ponta. Seu principal objetivo é facilitar a jornada do motorista, permitindo que ele encontre vagas disponíveis rapidamente por meio de navegação integrada e realize o check-in e o pagamento de forma ágil e simplificada.

Este repositório contém a camada de **Front-end** da aplicação, desenvolvida com tecnologias web nativas para garantir uma interface responsiva, rápida e intuitiva para o utilizador.

---

## ​⛓️​ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica das páginas e formulários.
* **CSS3:** Estilização, design responsivo (adaptável a telemóveis, tablets e computadores), uso de variáveis e animações.
* **JavaScript (ES6+):** Lógica de programação, manipulação dinâmica do DOM, validação de dados em formulários e integração/simulação de funcionalidades.
